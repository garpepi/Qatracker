###################
QATRACKER
###################

Only mini application that i create to help ake my job esier. This app trying to catch the dailyreport from compy outsource people about what they do in a day. This report have an effect to their performance metric and payment stuff.


*******************
Server Requirements
*******************

PHP version 5.4 or newer is recommended.

It should work on 5.2.4 as well, but we strongly advise you NOT to run
such old versions of PHP, because of potential security and performance
issues, as well as missing features.
